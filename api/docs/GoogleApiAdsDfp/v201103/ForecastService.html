Gets a Forecast on a prospective LineItem object. Valid

values for lineItemType are SPONSORSHIP and STANDARD. Other values will result in LINE_ITEM_TYPE_NOT_ALLOWED.

  • return: forecasted traffic estimates
  • access: public
the getForecast (lineItem $lineItem)

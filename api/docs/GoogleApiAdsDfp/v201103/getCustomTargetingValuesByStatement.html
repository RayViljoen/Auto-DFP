Class getCustomTargetingValuesByStatement

Description
Description | Vars (details) | Methods (details)

Gets a CustomTargetingValuePage of CustomTargetingValue objects that satisfy the given query.

The {@code WHERE} clause in the query must always contain customTargetingKeyId as one of its columns in a way that it is AND'ed with the rest of the query. So, if you want to retrieve values for a known set of key ids, valid query would look like:

  1. "WHERE customTargetingKeyId IN ('17','18','19')" retrieves all values that are associated with keys having ids 17, 18, 19.
  2. "WHERE customTargetingKeyId = '17' AND name = 'red'" retrieves values that are associated with keys having id 17 and value name is 'red'.
</p> The following fields are supported for filtering:

  • return: custom targeting values that match the given filter
